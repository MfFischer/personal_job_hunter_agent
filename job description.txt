Project Title

AI Personal Remote Job Hunter Assistant

📌 1️⃣ Project Overview

The AI Personal Remote Job Hunter Assistant is a private automation system hosted on your VPS that:

Automatically searches for 100% remote job opportunities daily

Filters roles based on your resume and preferences

Ranks job relevance using AI

Detects language (German vs English)

Selects the correct CV automatically

Generates tailored cover letters for the top matches

Sends a structured daily summary email

It is designed to:

Save time

Improve application quality

Avoid mass-application spam

Stay compliant with job platform policies

Operate at near-zero cost using free AI tiers

This is a personal productivity system first, with optional future SaaS potential.

🏗️ 3️⃣ System Architecture (High-Level)
Layer 1 — Job Discovery Engine

Function: Collect job listings daily.

Sources:

Public job board listings

Company career pages

RSS feeds

Remote job boards

Filters:

100% remote only

Relevant tech stack

Desired seniority level

Geographic compatibility (EU / global remote)

Output:

Raw job listings stored in database


No login automation. No credential scraping.

Layer 2 — Rule-Based Filtering (Cost-Free Layer)

Before using AI, apply deterministic filtering:

Keyword scoring (Python, AWS, TypeScript, SaaS, etc.)

Exclude “onsite” roles

Exclude junior/irrelevant seniority

Remove duplicates

Check already-applied history

Output:

Top ~20 candidate jobs per day


This keeps AI usage minimal.

Layer 3 — AI Relevance Scoring

Use free-tier AI (e.g., Gemini) to:

Analyze job description

Compare with structured resume profile

Generate:

Match score (0–100%)

Strength analysis

Missing skill insights

Language detection

Only applied to filtered jobs (not all scraped jobs).

Output:

Ranked job list with match insights

Layer 4 — Document Intelligence

For the Top 3 matches daily:

Detect job language

Select:

German CV (if job in German)

English CV (if job in English)

Generate tailored cover letter

Optionally adjust summary paragraph in CV

Optimization:

Use structured resume data to reduce token usage

Keep prompts short and focused

Layer 5 — Notification & Delivery

Send daily structured email via:

Hostinger SMTP (e.g., jobs@agosidentity.com
)

Email includes:

Top 3 jobs (with cover letters attached)

Next 7–10 ranked jobs (without cover letters)

Direct application links

Match score explanation

You manually review and apply.

🗃️ 4️⃣ Data Components
Database (Simple Schema)

Tables:

jobs (id, title, company, description, url, date_found)

job_scores (job_id, score, reasoning)

applications (job_id, applied_status, date_applied)

daily_reports (date, summary_sent)
